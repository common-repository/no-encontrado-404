Únase a nuestra red de colaboradores y sume su sitio a la búsqueda de niños perdidos. Usted puede hacer la diferencia.

== Description ==
Errores que se vuelven oportunidades

Al navegar por Internet es posible que en más de una oportunidad hayas llegado a una página que tiene el mensaje "error 404 - Página no encontrada". 
Esto ocurre cuando al navegar se intenta acceder a una url que no existe.
Las razones más comunes para que esto ocurra son: errores al tipear la dirección, enlaces rotos, resultados de búsqueda obsoletos y páginas eliminadas. 
¿Qué pasaría si pudieras aprovechar ese error y tiempo improductivo para darle un buen uso?

En esa oportunidad el usuario puede ver una foto de un chico perdido con el mensaje “Pagina no encontrada, pero podes ayudarnos a encontrar a ...”, aprovechando ese error en beneficio de toda la comunidad.  
Las imágenes e información de los niños se muestran de forma aleatoria de una base conformada con los registros oficiales de gobiernos y/u ONGs.

== Installation ==
Solo instalar el plugin y activar.
Con solo un par de click estará colaborando para que uno de estos niños vuelva a su hogar.
